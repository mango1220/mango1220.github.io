1. [217]Contain Duplicate

Given an array of integers, find if the array contains any duplicates. Your function should return true if any value appears at least twice in the array, and it should return false if every element is distinct.
给定一个整数数组，判断该数组是否有重复的数字。如果有数字出现了至少两次，那么你的函数应当返回true，如果每个数字都是不同的（唯一的），那么函数应当返回false。

方法1

利用STL中的set容器，因为set容器是单纯的key的集合，而且每个key都是唯一的，所以将题设数组存入set容器中，只会保留不同的元素。比较set的size和vector的size，若set较小说明有重复的元素被归为一个了，所以返回true，否则，set的size等于vector的size，返回false。
set是使用红黑树实现数据存储的。

2. [66]Plus One

Given a non-negative number represented as an array of digits, plus one to the number.
The digits are stored such that the most significant digit is at the head of the list.
将一组数字存入数组来代表一个非负数，对该数加1。
最高位的数字存在数组的第一个位置。

方法

检查从最低位开始，连续多少位是9，即检查出，从第几位开始不是9，在该位加1，之前为9的位数全部置0，若该位为最高位，则使用insert方法，添加一位，即新的最高位为1，其他位置0.

3. [189]Rotate Array

Rotate an array of n elements to the right by k steps.
For example, with n = 7 and k = 3, the array [1,2,3,4,5,6,7] is rotated to [5,6,7,1,2,3,4].
Note:
Try to come up as many solutions as you can, there are at least 3 different ways to solve this problem.
对一含有n个元素的数组向右翻转3步。
举例，n = 7， k = 3， 数组为[1,2,3,4,5,6,7]被翻转为[5,6,7,1,2,3,4]。

尽可能使用更多的方法，该问题至少有3种不同的解决方法。

将后k位的数组平移至数组头部，如例子中的5,6,7平移到1之前，5,6,7的顺序不变。涉及到数组的插入和删除。

5. [26]Remove Duplicates from Sorted Array

Given a sorted array, remove the duplicates in place such that each element appear only once and return the new length.
Do not allocate extra space for another array, you must do this in place with constant memory.
For example, Given input array nums = [1,1,2], Your function should return length = 2, with the first two elements of nums being 1 and 2 respectively. It doesn’t matter what you leave beyond the new length.
给定一个已排序的数组，删除重复的元素使得每个元素只出现一次，返回新的数组长度。不要另外新建一个数组，你必须在连续的内存上进行操作。
例如，给定一个数组[1,1,2]，你的函数必须返回长度为2，而且前两个元素必须分别为1和2。(……?)

方法

vector的erase方法删除某一元素后，其后面的所有元素的索引都会减一（相比未删除元素时的数组）。
对于一个已排序的数组，重复的元素只能存在于相邻的位置，所以只需要比较相邻两数是否相同，若相同则删除其中一个再进行比较。

6. [228]Summary Ranges

Given a sorted integer array without duplicates, return the summary of its ranges.
For example, given [0,1,2,4,5,7], return [“0->2”,”4->5”,”7”].
给定一个已排序数组，且该数组没有重复元素，返回其中元素范围的集合，即连续的一组数作为一个集合，范围为a->b。
例如，给定一个数组[0,1,2,4,5,7]，返回[“0->2”,”4->5”,”7”]。

方法

按序判断，从n=0开始，若第n个加一不等于第n+1个，则输出起始点0到n即”0->n”，再把n+1复制给n，起始点变为n+1，继续按前面的方法判断并输出，直到遍历全数组。
